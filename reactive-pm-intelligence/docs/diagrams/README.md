# Diagram Index

This directory contains editable draw.io source files for the solution.

- `system-architecture.drawio`: end-to-end system architecture and interaction surfaces.
- `service-class-diagram.drawio`: major service, route, and persistence relationships.
- `intelligence-flow.drawio`: analytics and PM generation execution flow.
- `business-value-flow.drawio`: business problem to value realization flow.

Open these files in diagrams.net or the VS Code draw.io extension.
